#  Shopping Basket

## MVVM + Flow Coordinator + RxSwift demo project

## Features:
- add a product to the basket
- remove a product from the basket
- display a total price in chosen currency

## Dependencies:
- RxSwift 4.4
- Alamofire 4.7
- fixer.io / exchangeratesapi.io

## Installation:
```
carthage update
```
